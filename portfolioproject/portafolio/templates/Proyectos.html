Taller Mecanico - Rayo Makween

Esta página web ha sido desarrollada para el taller mecánico "Rayo Makween". Ofrece una experiencia atractiva y fácil de navegar, permitiendo a los clientes explorar los servicios ofrecidos, conocer al equipo de mecánicos calificados y obtener información de contacto. Cuenta con un diseño responsivo que se adapta a diferentes dispositivos, asegurando una visualización óptima tanto en computadoras de escritorio como en dispositivos móviles.

Juego - Dinosaurio de Chrome

Este proyecto recrea el popular "Juego del Dinosaurio" de Google Chrome utilizando HTML y CSS. Mediante una interfaz minimalista pero cautivadora, el usuario puede controlar al icónico dinosaurio para esquivar obstáculos y acumular puntos. El diseño se ha realizado enteramente con elementos HTML y estilos CSS, demostrando la versatilidad y el potencial de estas tecnologías para crear experiencias interactivas atractivas.
